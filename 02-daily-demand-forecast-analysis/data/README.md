# Breast Cancer Wisconsin (Diagnostic) Data

This time-series data set was sourced from the [University of California Irvine Machine Learning Repository](https://archive.ics.uci.edu/dataset/409/daily+demand+forecasting+orders). The 12 features in this data were collected over a period of 60 days at a Brazilian logistics company. The data was created for a 2016 artificial neural network study by Ricardo Ferreira, Andrea Martiniano, Arthur Ferreira, Aleister Ferreira, and Renato Sassi.

Citation:

R. Ferreira, A. Martiniano, A. Ferreira, A. Ferreira, and R. Sassi. "Daily Demand Forecasting Orders," UCI Machine Learning Repository, 2016. [Online]. Available: [https://doi.org/10.24432/C5BC8T](https://doi.org/10.24432/C5BC8T).
