# Breast Cancer Wisconsin (Diagnostic) Data

This multivariate data set was sourced from the [University of California Irvine Machine Learning Repository](https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic). The 30 features in this data are computed from a digitized image of a fine needle aspirate of a breast mass. The data was created for a 1993 study by William Wolberg, Olvi Mangasarian, Nick Street, and W. Street.

Citation:

W. Wolberg, O. Mangasarian, N. Street, and W. Street. "Breast Cancer Wisconsin (Diagnostic)," UCI Machine Learning Repository, 1993. [Online]. Available: https://doi.org/10.24432/C5DW2B(https://doi.org/10.24432/C5DW2B).
